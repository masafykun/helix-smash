# 🔥 Helix Smash

> 片手操作でスパイラルタワーを回し、リングの隙間を狙って落ち続けるドロップアクション

スパイラルタワーを回転させて各リングの隙間を通し、クリーンなドロップを繋いでファイアボールを発動、プラットフォームを破壊しながらエンドレスに落下していくゲームです。暗いデンジャーアークを避けながら生き残ります。Unity で制作され、ブラウザ上でプレイできます。

## 🎮 操作方法

| 操作 | 動作 |
|---|---|
| ドラッグ / スワイプ | スパイラルタワーを回転させ、リングの隙間にボールを通す |

---

## ✨ 特徴

- **片手操作** — タワーを回すだけで遊べるシンプルな操作性
- **ファイアボール** — クリーンなドロップを繋ぐとファイアボールが発動しプラットフォームを破壊
- **エンドレス降下** — どこまで落下し続けられるかに挑戦
- **デンジャーアック回避** — 暗い危険ゾーンに触れないよう隙間を狙う
- **ブラウザプレイ** — WebGL ビルドにより GitHub Pages から直接プレイ可能
