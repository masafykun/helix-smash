# HELIX SMASH

One-finger helix drop: spin the spiral tower to thread each ring's gap, chain clean drops to ignite a fireball that smashes through platforms, and survive an endless descent while dodging the dark danger arcs.

**▶ Play in browser:** https://masafykun.github.io/helix-smash/

![screenshot](screenshot.png)

## About
A small game built with Unity (6000.0.77f1). The C# source is under `src/`.
This repository also hosts a WebGL build, playable directly in the browser via GitHub Pages.
